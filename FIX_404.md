API mengembalikan 404 "Halaman Tidak Ditemukan" karena:
1. **First-time deployment** memerlukan authorization manual
2. Web app access perlu di-aktifkan via UI

### Langkah 2: Authorize (Jika Muncul)

Jika muncul popup authorization:
1. Klik **Authorize access**
2. Pilih account Google (`dwisuliwa0019@gmail.com`)

5. Klik **Allow**

## 📞 Masih Bermasalah?

Jika masih 404 setelah authorize:

1. **Clear browser cache** (Ctrl+Shift+Delete)
2. **Incognito mode** dan coba lagi
3. **Check Executions** di Apps Script Editor untuk error detail
4. **Redeploy** dengan description baru
